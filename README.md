# Mushee Funding Hub (YC-style)

A premium black/red website for Mushee:

- **Overview**: product-grade landing page
- **Grants**: searchable funding database (Web3, AI startups, government, innovation programs, accelerators)
- **Commons**: public goods landing page with the tagline: **“Mushee is building within Commons”**

## Run locally

```bash
npm install
npm run dev
```

Open: http://localhost:3000

## Deploy (Vercel)

Push to GitHub → Import into Vercel → Deploy.

## Update the funding database

Edit:

- `data/grants.ts`

Replace the sample entries with real grants and links.
Keep it curated. Quality beats quantity.
